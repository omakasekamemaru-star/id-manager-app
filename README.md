# ID管理メモ

登録したWebサービスについて、「どのメールで登録したか」「どこにログイン情報を保存したか」「個人用か会社用か」を迷わないように管理するためのシンプルな静的Webアプリです。

## 注意

このアプリはパスワードを保存しません。メモ欄にもパスワードは書かないでください。

## 使い方

- `index.html` をブラウザで開くと使えます。
- HTML / CSS / JavaScriptだけで作られているため、GitHub Pagesで公開して利用できます。
- データはサーバーに送信せず、ブラウザの `localStorage` に保存します。
- 保存キーは `accountManagerRecords` です。

## データ保存について

`localStorage` は端末やブラウザごとに別々です。iPhone Safariで登録したデータは、PCのChromeや別の端末には自動共有されません。

大切なデータは、アプリ内のCSVエクスポート機能で定期的にバックアップすることをおすすめします。CSVインポートでは、既存データを消さずに追加できます。
